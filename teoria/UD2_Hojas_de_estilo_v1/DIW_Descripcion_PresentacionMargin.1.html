Descripción de las propiedades margin en elementos en bloque y en línea.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejemplo propiedad margin</title>
    
    <style>
            /* Definimos los márgenes de h1*/
            h1 {margin: 10px 50px 50px 50px;
                border: 1px black solid; }
            /* Definimos los márgenes de h2*/   
            h2 {margin: 50px 50px 50px 100px;
                border: 1px black solid; }
            /* Definimos los márgenes del span*/   
            span {margin: 5px 50px ;
                 border: 1px black solid; }
        
        
    </style>
</head>
<body>
    <h1> Título h1</h1> 
    <h2> Título h2</h1> 
    <h1> Título h1</h1> 
    <span> Elemento 1</span><span> Elemento 2</span>
</body>
</html>


Se muestra una pantalla  el código anterior, el resultado de la página y debajo de esta el depurador del navegador donde podemos ver las propiedades de los elementos seleccionados.

Seleccionamos un elemento h1 y le cambiamos los márgenes. Cambiados estos valores vamos al depurador y seleccionamos diferentes objetos y vemos sus propiedades, tamaño de caja, propiedades margin, padding y border.

Comprobamos también el colapso de los márgenes, vemos como los márgenes verticales de los elementos de caja se colapsan; el margen inferior de h1 es 50 igual que el margen superior de h2, estos se colapsan y el margen de ambos es 50, es decir se queda con el mayor de los dos.

Posteriormente vamos a los elementos de línea  y vemos que los márgenes laterales no se colapsan y se suman, vemos como la diferentecia lateral de dos elementos es igual al margen derecho de un elemento más el margen izquierdo del elemento siguiente.
